# SPOT
SPOT stands for Spatial Planning and Optimization Technologies
-------------------------------------------------------------
The project aims to plan urban land, infrastructure, facilities, transportation, utilities, or resources across geographic space using operations research, mathematical optimization, ans multi-agent simulation.

All Java sources are available. They should be included in a java project containing MATSim library. 

A jar file onf the projet (SPOT-0.0.1;jar) is available here :

https://filesender.renater.fr/?s=download&token=efbaa508-ae6b-4c8b-a216-acd86668336e

A simulation can executed as follows : java -jar SPOT-0.0.1.jar "your matsim xml files directory"

"your matsim xml files directory" is the directory where all xml files used for the simulation belongs : "config.xml","network.xml.gz",plans.xml","facilities.xml"
